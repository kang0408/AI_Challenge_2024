# HIT_ChiYeuMinhEm

Giao diện website AI Chanllenge CLB Tin học HIT

## Hướng dẫn

### 1. Tạo file .env dựa vào file mẫu (.env.example)

- Chú thích

```
VITE_API_URL=http://localhost:3000 //Base URL API
```

### 2. Tải Dependencies

```bash
npm i
# hoặc
yarn
```

### 3. Chạy ứng dụng (development):

```bash
npm run dev
# hoặc
yarn dev
```
